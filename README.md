# micro-service
  Dubbo + Kubernetes 微服务容器化实践示例项目
  
  # 主要业务功能：
    - 用户服务
        -- 用户登录
        -- 用户注册
        -- 用户基本信息查询
        -- 无状态，无session
        -- 单点登录
    - 课程服务
        -- 登录验证
        -- 课程的CRUD
    - 信息服务
        -- 发送邮件
        -- 发送短信
    - 用户 edge-service
    - 课程 edge-service
    - API Gateway
